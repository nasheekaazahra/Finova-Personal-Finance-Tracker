## ✨ Overview

**Finova** is a modern personal finance tracker designed to help users manage their finances with ease. It provides an intuitive dashboard for tracking income and expenses, setting monthly budgets, monitoring savings goals, and analyzing financial health through interactive charts and insights.

The application enables users to record daily transactions, categorize expenses, plan spending limits, and visualize their financial activities in real time. With features such as **Budget Planning**, **Savings Goals**, **Financial Health Score**, **Interactive Analytics**, **PDF Report Export**, and **Dark/Light Mode**, Finova offers a clean and user-friendly experience for better financial management.

Built with a responsive and elegant interface, Finova is designed to deliver a seamless experience across desktop, tablet, and mobile devices, making personal finance management simple, organized, and accessible anytime.


---

## Highlights

- 💰 Track income and expense transactions
- 📊 Interactive financial dashboard with analytics
- 🎯 Savings goal management with progress tracking
- 📅 Monthly budget planning by category
- ❤️ Financial Health Score based on spending and savings
- 📈 Expense breakdown by category with charts
- 📄 Export financial reports to PDF
- 🌙 Dark & Light mode support
- 🔍 Search, filter, edit, and delete transactions
- 📱 Fully responsive for desktop, tablet, and mobile
- 💾 Data persists locally via `localStorage` — no server required
